# CancerianMind 🧠

**Your Psychology AI Companion** — A full-featured mental wellness web app.

## Features
- 💬 AI Therapy Chat (CBT, mindfulness, crisis support)
- 📋 Psychological Assessments (PHQ-9, GAD-7, PSS-10)
- 📖 Psychology Education Library
- 📊 Mood Dashboard & Tracking

## Deploy on Vercel

### Option 1 — Drag & Drop (Easiest)
1. Go to [vercel.com/new](https://vercel.com/new)
2. Drag the entire `cancerianmind` folder into the upload area
3. Click **Deploy** — done!

### Option 2 — GitHub + Vercel (Recommended)
1. Create a new GitHub repository
2. Upload all these files to the repo
3. Go to [vercel.com/new](https://vercel.com/new)
4. Import your GitHub repo
5. Vercel auto-detects settings — click **Deploy**
